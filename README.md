## Development

### Run locally

- Clone the Repo

  ```
  git clone https://github.com/gurupawar/atDev-assignment.git

  cd atDev-assignment
  ```

- Import project in Eclipse / Vs code / IntelliJ Idea and navigate to `main > java > com > guru > AtDev-Assignment > AtDevSssigmentApplication.java`

and run the project
